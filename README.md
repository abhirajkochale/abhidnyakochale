# Abhidnya Kochale — Portfolio

React + Vite portfolio. Tech stack:
- **React 18** + **Vite 5**
- **GSAP** (ScrollTrigger)
- **Lenis** smooth scroll
- **React Router DOM v6**

## Fonts
- Caveat 400/700 — display/headings
- Space Mono 400/700 — body/captions
- Cormorant Garamond italic 300 — poetic pull-quotes

## Dev
```bash
npm install
npm run dev
```
